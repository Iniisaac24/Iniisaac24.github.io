Property Price
Prediction & Mapping

XGBoost regression on King County housing data with an interactive map identifying undervalued neighbourhoods for investment.

The Problem

Finding undervalued properties in a noisy market

A boutique real estate investment firm evaluates residential properties in King County, Washington. With 21,000+ recent transactions in the dataset, the challenge isn't access to data — it's extracting signal from noise. The team needs a model that can estimate fair market value for any property, and a map that visually highlights where actual listing prices fall below that estimate.

An undervalued property — one where the model predicts higher value than the asking price — represents a potential acquisition opportunity. The map makes this actionable for non-technical investment managers.

Outcome

XGBoost model achieved R² of 0.88 on the holdout set. Two ZIP code clusters identified with consistent 12–18% undervaluation relative to comparable properties — flagged as priority acquisition targets in the interactive map.

Approach

Regression model → residuals → investment map

01

Exploratory Analysis

Sale price is right-skewed — log-transformed for modelling. Strong correlations with sqft_living (r=0.70), grade (r=0.67), and bathrooms (r=0.52). Properties on the waterfront command a 53% price premium on average.

02

Geospatial Visualisation

Plotted all 21,000 properties on a folium map coloured by price per square foot. Immediate visual confirmation of the Seattle price gradient — prices peak in Medina and drop toward the county edges.

03

Feature Engineering

Created: price_per_sqft, age_of_home (2024 − yr_built), renovation_recency (years since last renovation), basement_flag, distance_from_seattle_centre (via geopy Haversine). Binned neighbourhoods into price quintiles.

04

Baseline: Linear Regression

Ridge regression on log-transformed price. R²=0.70 — good interpretability, establishes which features have positive and negative coefficients. Used to explain the direction of each feature's effect to non-technical stakeholders.

05

XGBoost Regression

Trained with 5-fold cross-validation. Permutation importance used to reduce to top 20 features. Final model R²=0.88 on holdout, RMSE=$62,400. Residuals plotted — no systematic bias by neighbourhood after log transformation.

06

Undervaluation Map

Computed residual = predicted price − actual price. Properties where residual > +15% marked green on the folium map (undervalued). Properties where residual < −15% marked red (overvalued). Saved as interactive HTML file.

Code Sample

Building the undervaluation map

import folium
import numpy as np

# Compute residuals
df['predicted_price'] = np.expm1(model.predict(X_test))
df['actual_price']    = np.expm1(y_test)
df['residual_pct']    = (df['predicted_price'] - df['actual_price']) / df['actual_price']

# Build folium map centred on King County
m = folium.Map(location=[47.5, -122.0], zoom_start=10, tiles='CartoDB positron')

for _, row in df.iterrows():
    undervalued = row['residual_pct'] > 0.15
    overvalued  = row['residual_pct'] < -0.15

    color = '#22c55e' if undervalued else '#ef4444' if overvalued else '#94a3b8'

    folium.CircleMarker(
        location=[row['lat'], row['long']],
        radius=4,
        color=color,
        fill=True,
        fill_opacity=0.65,
        popup=folium.Popup(
            f"Actual: ${row['actual_price']:,.0f}
"
            f"Predicted: ${row['predicted_price']:,.0f}
"
            f"Gap: {row['residual_pct']:+.1%}",
            max_width=200
        )
    ).add_to(m)

m.save('outputs/property_map.html')

Key Findings

What the model uncovered

sqft_living is king — but grade matters more at the top

Below $600K, square footage is the dominant price driver. Above $600K, the property grade score (quality of construction and design) becomes the strongest predictor, surpassing size.

Two undervalued ZIP clusters identified

ZIP codes 98002 and 98023 (Auburn and Federal Way areas) showed consistent 12–18% undervaluation across 300+ properties — properties with equivalent features to comparable ZIPs priced significantly lower.

Renovations within 10 years add ~8% to predicted value

Properties renovated within the last decade commanded an 8% premium on average, even controlling for age. Unrenovated properties in otherwise high-value areas may represent upside opportunity.

Waterfront premium is real but concentrated

Waterfront properties averaged 53% higher prices, but the premium ranged from 20% to 180% depending on view quality and lot size. The model captures this via interaction between waterfront flag and sqft_lot.

Distance from Seattle centre decays non-linearly

Price drops sharply in the first 10 miles from the city centre, then flattens. Properties 15–25 miles out are priced similarly to those 25–40 miles out — the suburban pricing plateau.
