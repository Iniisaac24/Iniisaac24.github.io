The Problem

High readmission rates carry real penalties

Under the CMS Hospital Readmissions Reduction Program, hospitals are penalised financially for excess 30-day readmission rates in key conditions including heart failure, pneumonia, and diabetes. For a regional hospital network, even a 1% reduction in readmission rate can mean significant cost savings and better patient outcomes.

The clinical analytics team needed to understand which patient profiles carry the highest readmission risk so care coordinators can prioritise follow-up interventions at discharge.

The ask: Analyse the patient population, identify the top risk factors for 30-day readmission, and deliver findings in a format the clinical team can act on — not just a list of p-values.

Process

Cleaning 50 columns, finding 5 signals

01

Missing value handling

This dataset uses '?' as a missing value marker — not NaN. Replaced all instances before any analysis. Dropped columns with over 40% missing data: weight, payer_code, and medical_specialty. Retained 47 usable features.

02

Target definition

Readmission column has three values: <30, >30, and NO. Collapsed to a binary target: <30 days = 1 (clinically actionable), everything else = 0. Overall 30-day rate: 11.2%.

03

Diagnosis categorisation

The primary diagnosis column contains raw ICD-9 codes — over 900 unique values. Applied a mapping function to group them into 8 clinical categories: Circulatory, Respiratory, Digestive, Diabetes, Injury, Musculoskeletal, Genitourinary, and Other.

04

Statistical testing

Chi-square tests on all categorical variables against the binary readmission target. Mann-Whitney U tests (non-parametric, appropriate for skewed count variables) on continuous variables including length of stay, number of medications, and prior inpatient visits.

05

Visualisation

Heatmap of readmission rate by age group × diagnosis category — 10 age bands × 8 diagnosis categories — surfaces the highest-risk patient intersection clearly. All findings framed as clinical recommendations, not statistical summaries.

ICD-9 code categorisation

def diag_category(code):
    try:
        code = str(code)
        if code.startswith('V') or code.startswith('E'): return 'Other'
        n = float(code)
        if 390 <= n <= 459 or n == 785: return 'Circulatory'
        if 460 <= n <= 519 or n == 786: return 'Respiratory'
        if 520 <= n <= 579 or n == 787: return 'Digestive'
        if n == 250:                    return 'Diabetes'
        if 800 <= n <= 999:             return 'Injury'
        return 'Other'
    except:
        return 'Other'

def chi_square_test(df, col, target='readmitted_30'):
    ct = pd.crosstab(df[col].dropna(), df.loc[df[col].notna(), target])
    chi2, p, dof, _ = chi2_contingency(ct)

Findings

Five recommendations for the care team

Patients with circulatory diagnoses are 1.4× more likely to be readmitted within 30 days compared to the overall population. Clinical follow-up protocols for discharge from circulatory episodes should be reviewed — particularly around medication reconciliation.

Prior inpatient visits are the strongest predictor of readmission. Patients with 2 or more prior inpatient stays in the past year are readmitted at nearly 3× the rate of first-time patients. A case management flag for high-frequency utilizers at discharge would catch the majority of preventable readmissions.

Polypharmacy is significantly associated with readmission (Mann-Whitney U, p < 0.001). Patients on above-median medication counts at discharge have a 14% higher readmission rate. A pharmacist-led medication review at discharge for patients on 15+ medications is recommended.

The 70–80 age group with circulatory or respiratory diagnoses represents the single highest-risk intersection in the age × diagnosis heatmap — readmission rates exceeding 15% for this cohort. Targeted post-discharge phone follow-up at day 3 and day 7 for this group is supported by the data.

Discharge disposition matters. Patients discharged to home without home health have statistically higher readmission than those with home health arranged. Expanding home health referrals at discharge for high-risk patients is the highest-leverage structural intervention identified.
